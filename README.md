# 🌵 Old Arizona — 11.04.2026

Набор Lua-скриптов и модификаций для GTA: San Andreas на сервере **Arizona RP (SA-MP)**.  
Использует **MoonLoader** и **ModLoader** для загрузки и управления скриптами в игре.

Этот билд возвращает классический облик сервера: старые скины полиции, оригинальные анимации и адекватный баланс наличных — без мультяшного оружия и прочего визуального мусора.

---

## ✨ Что изменено

- 👮 **Скины PD** — возвращены оригинальные скины полицейских департамента
- 🔫 **Оружие** — убраны мультяшные/аркадные модели, восстановлены классические
- 💵 **Наличные** — возвращён разумный баланс отображения денег
- 🎬 **Анимации полиции** — восстановлены оригинальные полицейские анимации

---

## 📁 Структура репозитория


---

## ⚙️ Требования

- **GTA: San Andreas** (версия 1.0 US)
- **SA-MP** (клиент)
- **MoonLoader** — загрузчик Lua-скриптов для GTA SA
- **ModLoader** — загрузчик модификаций без замены файлов игры
- **SAMPFUNCS** (рекомендуется, для расширенного функционала)

---

## 🚀 Установка

1. Убедитесь, что у вас установлены MoonLoader и ModLoader.
2. Скопируйте содержимое папки `moonloader/` в директорию `moonloader/` вашей игры.
3. Скопируйте содержимое папки `modloader/` в директорию `modloader/` вашей игры.
4. Поместите `samp-cursor-patch.asi` в корневую папку GTA: San Andreas.
5. Запустите игру и подключитесь к серверу Arizona RP.
